# Edge 11ty Presenter #

An [11ty](https://www.11ty.dev/) website showing presentation content from Sitecore Experience Edge and managed in Content Hub ONE.

# Getting Started #

## Requirements ##

This project requires minimum Node version 14.

## Steps ##

1. Restore the NPM packages.

    npm install

1. Run 11ty to generate the static site

    npx @11ty/eleventy
